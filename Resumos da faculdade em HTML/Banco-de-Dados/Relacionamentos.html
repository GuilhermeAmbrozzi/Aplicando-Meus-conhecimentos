Relacionamento

Entendemos que as entidades não podem ficar isoladas, uma vez que as informações estarão organizadas futuramente para o acesso de forma integrada.Para uma organização sem a perda de informações, as entidades devem estar associadas, ligadas entre si. No MER entenda que não é permitido ligar uma entidade a outra, por isso, quando é uma associação ela é representada por um Relacionamento que é representado nos documentos em forma de um losango. Segue exemplo de como as entidades são ligadas através de um relacionamento.

Para ficar mais fácil de definir um relacionamento entra duas entidades, devemos verificar se há correlação entre elas e podemos fazer isso colocando um Verbo para tentar associá-las. É importante analisar se a associação entre as entidades é verdadeiras em ambos os sentidos.
